To Add Awesome Arabic Fonts To Your Web Site Add This Code In Befor Close <head> Tag
--------------------------------------------------------------------------------------

For Font1 Add This
	<link rel="stylesheet" type="text/css" href="https://bit.ly/Font1IQ4UStudio">
--------------------------------------------------------------------------------------

For Font2
Add This
	<link rel="stylesheet" type="text/css" href="https://bit.ly/Font2Iq4ustudio">
  --------------------------------------------------------------------------------------

  
  In Css Page For Font1 Add This To Class You Want To Change Font In which:
  
    font-family: 'Changa', sans-serif;
--------------------------------------------------------------------------------------
  In Css Page For Font2 Add This To Class You Want To Change Font In which:
  
    font-family: 'Tajawal', sans-serif;
